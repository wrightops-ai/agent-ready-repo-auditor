# Agent-Ready Repo Fix Plan

Turn one free automated audit into three exact, human-reviewed fixes.

## Fixed scope

For **$149 USD**, WrightOps reviews one completed
[free public audit](https://github.com/wrightops-ai/agent-ready-repo-auditor/issues/new?template=audit-request.yml)
and posts exactly three fix cards to that same public GitHub issue.

Each fix card includes:

1. the evidenced repository-readiness gap and immutable public source links;
2. the exact file or path to change;
3. a bounded change outline;
4. the verification or acceptance check; and
5. relevant limitations and risky-action boundaries.

Delivery is within **one business day** after settled payment and receipt of all
required public inputs. The review covers one public GitHub repository at one
immutable default-branch revision and includes at most 45 minutes of human
review.

The buyer must be authorized to commission a public review of the repository.

records and transaction identifiers remain private and are never posted to the
GitHub issue.

## Exclusions

The Fix Plan does not include:

- code or documentation changes;
- private repositories or private data;
- credentials, production access, deployment, or infrastructure work;
- vulnerability testing or a security, privacy, legal, or compliance review;
- a live meeting or implementation handoff;
- validation of undocumented claims; or
- a guaranteed score, merge, outcome, savings, or operational improvement.

Missing or ambiguous evidence remains missing. The plan identifies a bounded
way to address the evidenced gap; it does not represent the repository as safe,
secure, compliant, or ready for every agent workflow.
